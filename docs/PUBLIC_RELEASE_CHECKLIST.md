# Public release checklist

- [ ] Rotate and scan for credentials, share URLs, signing material, and private fixtures.
- [ ] Enable GitHub secret scanning, push protection, Dependabot, and private vulnerability reporting.
- [ ] Confirm MIT license and dependency/license inventory.
- [ ] Validate `PrivacyInfo.xcprivacy`, App Store privacy answers, privacy policy, and deletion language.
- [ ] Provision the production CloudKit container and deploy development schema to production.
- [ ] Test invite, accept, decline, revoke, leave, offline retry, and Apple ID switch on two physical devices.
- [ ] Test JSON/CSV export and representative TV Time imports with rollback before apply.
- [ ] Audit VoiceOver, Dynamic Type accessibility sizes, increased contrast, reduced motion/transparency, and button shapes.
- [ ] Verify TMDB and JustWatch attribution and all official cinema links.
- [ ] Run CI on the release commit and archive with release signing.
- [ ] Remove private roadmap material and confirm issue/discussion templates are public-safe.
